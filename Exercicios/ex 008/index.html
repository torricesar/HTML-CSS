Principais formatações

Negrito / Destaque

Para usar o negrito, use a tag B

Nessa frase temos termo em destaque unsando a tag STRONG

Itálico / Ênfase

Usamos a tag I para usar o itálico

Usamos a tag EM para usar o itálico semantico

Texto marcado

Podemos também colocar um marcador de texto usando a tag MARK

Se quiser mudar a cor do marcador acrecentamos em frente da tag MARK a palavra STYLE daí vai em blackground-color

Agora vamos fazer um texto grande e outro pequeno

Quando usamos a tag SMALL o texto fica pequeno

Texto deletado

Podemos marcar um texto como excluído e não precisa ser considerado, para isso usamos a tag DEL desse jeito

Podemos também colocar um texto como inserido usando a tag INS dessa forma,

Textos sobrescritos

Para inserir coisas do tipo x20+3 usamos a tag SUP x+3

Ou usamos para inserir letras ou numeros na parte iferior a tag SUB tipo H2O H2O
